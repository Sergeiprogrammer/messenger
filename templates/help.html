Добро пожаловать в Безопасный Мессенджер

В этом мессенджере общение возможно только при условии, что обе стороны знают Уникальный ID друг друга. Это гарантирует, что сообщения не могут быть отправлены без правильной идентификации.

Ваш Уникальный ID

При регистрации вам присваивается Уникальный ID, который вы должны использовать для связи с другими. Берегите этот ID!

Отправка Сообщений

Для отправки сообщения вам нужны:

  • Ваш Уникальный ID
  • ID Получателя

Без этих ID отправка сообщений невозможна. Также чтобы видеть новые сообщения собеседника нажмите на кнопку включить атообновления а когда будете писать текст нажмите выключить автообновление
